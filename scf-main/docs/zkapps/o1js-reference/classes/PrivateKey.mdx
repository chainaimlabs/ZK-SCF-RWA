### random()

```ts
static random(): PrivateKey
```

Generate a random private key.

You can obtain the associated public key via [toPublicKey](PrivateKey.mdx#topublickey).
And generate signatures via [Signature.create](Signature.mdx#create).

Note: This uses node or browser built-in APIs to obtain cryptographically strong randomness,
and can be safely used to generate a real private key.

#### Returns

[`PrivateKey`](PrivateKey.mdx)

a new [PrivateKey](PrivateKey.mdx).
