### verify()

```ts
static verify(
   publicInput: any[], 
   verificationKey: VerificationKey, 
proof: Proof): Promise<boolean>
```

Verifies a proof using the public input, the proof, and the initial [Keypair](Keypair.mdx) of the circuit.

#### Parameters

• **publicInput**: `any`[]

• **verificationKey**: `VerificationKey`

• **proof**: `Proof`

#### Returns

`Promise`\<`boolean`\>

#### Example

```ts
const keypair = await MyCircuit.generateKeypair();
const proof = await MyCircuit.prove(privateInput, publicInput, keypair);
const isValid = await MyCircuit.verify(publicInput, keypair.vk, proof);
```
